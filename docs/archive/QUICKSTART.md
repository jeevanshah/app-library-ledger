# Quick Start Guide — App Library Ledger

## Run the App

### Step 1: Navigate to the project
```bash
cd app_library_ledger
```

### Step 2: Get dependencies
```bash
flutter pub get
```

### Step 3: Run on your device/emulator

**iOS (on Mac):**
```bash
flutter run
```

**Android (with emulator running):**
```bash
flutter run
```

**Windows (for testing):**
```bash
flutter run -d windows
```

## Using the App

### Add Your First App

1. Tap **+** button (bottom right)
2. Enter:
   - **App Name** — e.g., "Notion"
   - **App Store Link** — e.g., "https://apps.apple.com/app/notion-notes-planner/id1232780281"
   - **Category** — pick from list or create custom
   - **Notes** — optional, e.g., "Main note-taking app"
3. Tap **Save App**

### View Your Library

- **List** — All apps with colored category dots
- **Search** — Find apps by name
- **Filter** — Tap category chips to filter
- **Count** — See how many apps in each category (spot duplicates!)

### Manage Categories

1. Tap **≡** (category icon) in top right
2. View all categories
3. Rename custom categories
4. Delete custom categories (default ones can't be deleted)

### Delete an App

1. Swipe left or tap trash icon on any app
2. Confirm deletion

## Key Takeaways

**Purpose:** Catalog apps by category and spot duplicates  
**Privacy:** 100% offline, no account required  
**Speed:** Add an app in 10 seconds  
**Customizable:** Create your own categories
